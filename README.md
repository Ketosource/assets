# Ketosource Assets

Shared image and asset hosting for Ketosource documents (Notion, presentations, etc.).

## Usage

Upload files to the appropriate folder, then use the raw GitHub URL in Notion or other tools:

```
https://raw.githubusercontent.com/Ketosource/assets/main/diagrams/your-image.png
```

## Folder Structure

| Folder | Contents |
|--------|----------|
| `diagrams/` | Value chains, org charts, process maps, architecture diagrams |
| `charts/` | Data visualizations, graphs |
| `covers/` | Notion page covers, banners |
| `logos/` | Brand logos, icons |

## How to Add Images

1. Navigate to the folder in GitHub
2. Click "Add file" > "Upload files"
3. Drag and drop your image
4. Commit directly to `main`
5. Copy the raw URL for use in Notion
